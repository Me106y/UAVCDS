API参考文档

大疆航线规划MCP服务的完整API参考，包括所有工具的详细接口说明。

概述

大疆航线规划MCP服务基于Model Context Protocol (MCP)标准，提供了一套完整的无人机航线规划API。

基本信息

协议版本 MCP 1.0
服务版本 1.0.0
通信方式 JSON-RPC over stdio
编码格式 UTF-8

支持的功能

  • ✅ Tools - 10个专业航线规划工具
  • ❌ Resources - 暂不支持资源访问
  • ❌ Prompts - 暂不支持提示模板
  • ✅ Logging - 完整的日志记录

认证

当前版本的MCP服务运行在本地环境，通过stdio通信，无需额外认证。

连接方式

# 直接启动服务
python run_server.py

# 或通过MCP客户端连接
{
  "command": "python",
  "args": ["run_server.py"],
  "env": {
    "PYTHONPATH": ".",
    "LOG_LEVEL": "INFO"
  }
}

未来版本

未来版本将支持以下认证方式：

  • API Key认证

数据类型

基础类型

Coordinate (坐标)

{
  "latitude": number,   // 纬度 (-90 到 90)
  "longitude": number,  // 经度 (-180 到 180)
  "altitude": number    // 高度 (米，可选)
}

Waypoint (航点)

{
  "latitude": number,
  "longitude": number,
  "altitude": number,
  "actions": Action[],  // 航点动作列表
  "speed": number,      // 航点速度 (可选)
  "heading": number     // 航向角 (可选)
}

Action (动作)

{
  "type": string,       // 动作类型
  "parameters": object  // 动作参数 (可选)
}

// 支持的动作类型:
// - "take_photo": 拍照
// - "start_record": 开始录像
// - "stop_record": 停止录像
// - "gimbal_pitch": 云台俯仰
// - "hover": 悬停
// - "rotate_aircraft": 飞机旋转

SurveyArea (测区)

{
  "type": "rectangle" | "polygon",
  "coordinates": Coordinate[]
}

// 矩形: [左下角, 右上角]
// 多边形: [点1, 点2, ..., 点1] (闭合)

FlightParams (飞行参数)

{
  "speed": number,              // 飞行速度 (m/s)
  "altitude": number,           // 飞行高度 (m)
  "heading_mode": string,       // 航向模式
  "turn_mode": string,          // 转弯模式
  "gimbal_pitch": number        // 云台俯仰角 (可选)
}

响应类型

ToolResponse (工具响应)

{
  "success": boolean,
  "message": string,
  "data": object,       // 具体数据 (成功时)
  "error": string,      // 错误信息 (失败时)
  "error_code": string  // 错误代码 (失败时)
}

错误处理

错误响应格式

{
  "success": false,
  "error": true,
  "message": "错误描述信息",
  "error_code": "ERROR_CODE",
  "details": {
    "parameter": "参数名",
    "value": "参数值",
    "expected": "期望值"
  },
  "suggestions": [
    "解决建议1",
    "解决建议2"
  ]
}

常见错误代码

错误代码 描述 解决方案
VALIDATION_ERROR 参数验证失败 检查输入参数格式和范围
AIRCRAFT_COMPATIBILITY_ERROR 设备兼容性错误 选择支持的设备型号
COORDINATE_ERROR 坐标系统错误 检查坐标格式和范围
PERFORMANCE_LIMITATION 性能限制 调整任务参数或分段执行
FILE_GENERATION_ERROR 文件生成失败 检查输出目录权限

工具列表

plan_waypoint_mission

功能: 规划航点飞行任务

// 请求参数
{
  "waypoints": Waypoint[],
  "aircraft_type": string,
  "flight_params": FlightParams
}

// 响应数据
{
  "success": true,
  "data": {
    "flight_path": {
      "waypoint_count": number,
      "total_distance": number,
      "estimated_flight_time": number
    },
    "mission_config": {
      "aircraft_type": string,
      "flight_speed": number,
      "flight_altitude": number
    },
    "waypoints": Waypoint[]
  }
}

plan_mapping_mission

功能: 规划建图航拍任务

// 请求参数
{
  "survey_area": SurveyArea,
  "mapping_params": {
    "altitude": number,
    "overlap_rate": {
      "front": number,
      "side": number
    },
    "direction": number,
    "margin": number
  },
  "aircraft_type": string
}

// 响应数据
{
  "success": true,
  "data": {
    "flight_lines": number,
    "total_waypoints": number,
    "coverage_area": number,
    "estimated_photos": number,
    "flight_distance": number,
    "flight_time": number,
    "waypoints": Waypoint[]
  }
}

optimize_route

功能: 优化航线路径

// 请求参数
{
  "waypoints": Waypoint[],
  "optimization_method": "nearest_neighbor" | "genetic_algorithm" | "simulated_annealing" | "two_opt" | "christofides",
  "optimization_objective": "minimize_distance" | "minimize_time" | "minimize_energy" | "balanced",
  "flight_speed": number,
  "start_point_fixed": boolean,
  "max_iterations": number
}

// 响应数据
{
  "success": true,
  "data": {
    "optimized_waypoints": Waypoint[],
    "optimized_distance": number,
    "original_distance": number,
    "distance_saved": number,
    "time_saved": number,
    "optimization_stats": {
      "method": string,
      "iterations": number,
      "execution_time": number
    }
  }
}

generate_kmz

// 请求参数

{

SDK和客户端库

Python SDK

from mcp_client import MCPClient

# 创建客户端
client = MCPClient("dji-waypoint-mcp")

# 调用工具
result = await client.call_tool(
    "plan_waypoint_mission",
    {
        "waypoints": [...],
        "aircraft_type": "M30",
        "flight_params": {...}
    }
)

print(result)

JavaScript SDK

import { MCPClient } from 'mcp-client-js';

// 创建客户端
const client = new MCPClient('dji-waypoint-mcp');

// 调用工具
const result = await client.callTool('plan_waypoint_mission', {
    waypoints: [...],
    aircraft_type: 'M30',
    flight_params: {...}
});

console.log(result);

REST API封装

// 使用HTTP客户端
const response = await fetch('/api/missions', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        type: 'waypoint',
        config: {
            waypoints: [...],
            aircraft_type: 'M30'
        }
    })
});

const result = await response.json();
