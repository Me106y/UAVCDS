MCP工具详细文档

大疆航线规划MCP服务提供了10个专业的航线规划工具，每个工具都针对特定的飞行场景进行了优化。

工具概览

工具名称 功能描述 适用场景 支持机型
plan_waypoint_mission 航点飞行任务规划 自定义航点飞行、精确定位作业 全系列
plan_mapping_mission 建图航拍任务规划 地形测绘、正射影像采集 全系列
plan_oblique_mission 倾斜摄影任务规划 三维建模、倾斜摄影测量 M300/M350系列
plan_strip_mission 航带飞行任务规划 线性巡检、管道巡查 全系列
optimize_route 航线路径优化 路径优化、飞行效率提升 全系列
coordinate_multi_flights 多航线协调 复杂任务、多机协同 全系列
query_device_info 设备信息查询 设备兼容性检查 全系列
validate_mission_compatibility 任务兼容性验证 安全检查、参数验证 全系列
utility_functions 辅助功能工具 坐标转换、距离计算 全系列
generate_kmz KMZ文件生成 航线文件导出 全系列

1. 航点飞行任务规划 (plan_waypoint_mission)

规划自定义航点飞行任务，支持精确的航点定位和动作配置。适用于需要精确控制飞行路径和拍摄点的场景。

输入参数

参数 类型 必填 说明
waypoints Array ✅ 航点列表，至少2个航点
aircraft_type String ✅ 无人机型号 (M30, M300_RTK等)
flight_params Object ✅ 飞行参数配置

航点参数结构

{
  "latitude": 39.9042,        // 纬度 (-90 到 90)
  "longitude": 116.4074,      // 经度 (-180 到 180)
  "altitude": 100.0,          // 高度 (米)
  "actions": [                // 航点动作列表
    {
      "type": "take_photo",   // 动作类型
      "parameters": {         // 动作参数
        "suffix": "wp_001"
      }
    }
  ]
}

支持的航点动作

  • take_photo - 拍照
  • start_record - 开始录像
  • stop_record - 停止录像
  • gimbal_pitch - 云台俯仰角调整
  • hover - 悬停等待
  • rotate_aircraft - 飞机旋转

使用示例

{
  "waypoints": [
    {
      "latitude": 39.9042,
      "longitude": 116.4074,
      "altitude": 100.0,
      "actions": [
        {
          "type": "take_photo",
          "parameters": {"suffix": "point_01"}
        }
      ]
    },
    {
      "latitude": 39.9142,
      "longitude": 116.4174,
      "altitude": 100.0,
      "actions": [
        {
          "type": "gimbal_pitch",
          "parameters": {"angle": -45}
        },
        {
          "type": "take_photo",
          "parameters": {"suffix": "point_02"}
        }
      ]
    }
  ],
  "aircraft_type": "M30",
  "flight_params": {
    "speed": 5.0,
    "altitude": 100.0,
    "heading_mode": "follow_wayline",
    "turn_mode": "to_point_stop"
  }
}

2. 建图航拍任务规划 (plan_mapping_mission)

自动生成建图航拍任务，支持矩形和多边形测区的平行航线规划。适用于地形测绘、正射影像采集等场景。

测区类型

矩形测区

{
  "type": "rectangle",
  "coordinates": [
    [116.4074, 39.9042],  // 左下角
    [116.4174, 39.9142]   // 右上角
  ]
}

多边形测区

{
  "type": "polygon",
  "coordinates": [
    [116.4074, 39.9042],
    [116.4174, 39.9042],
    [116.4174, 39.9142],
    [116.4074, 39.9142],
    [116.4074, 39.9042]  // 闭合多边形
  ]
}

重叠率配置建议

应用场景 航向重叠率 旁向重叠率 说明
一般测绘 70-80% 60-70% 平衡质量和效率
高精度测绘 80-85% 70-75% 更好的拼接效果
快速巡查 60-70% 50-60% 提高飞行效率

3. 倾斜摄影任务规划 (plan_oblique_mission)

生成倾斜摄影任务，自动规划五个方向的航线（正射+四个倾斜方向）。适用于三维建模、倾斜摄影测量等场景。

倾斜角度配置

推荐倾斜角度

  • -30° - 适用于低层建筑
  • -45° - 通用角度，适用于大多数场景
  • -60° - 适用于高层建筑

倾斜方向

  • 0° - 北向倾斜
  • 90° - 东向倾斜
  • 180° - 南向倾斜
  • 270° - 西向倾斜

完整配置示例

{
  "survey_area": {
    "type": "polygon",
    "coordinates": [
      [116.4074, 39.9042],
      [116.4174, 39.9042],
      [116.4174, 39.9142],
      [116.4074, 39.9142],
      [116.4074, 39.9042]
    ]
  },
  "oblique_params": {
    "altitude": 150.0,
    "overlap_rate": {
      "front": 0.8,
      "side": 0.7
    },
    "oblique_angle": -45.0,
    "directions": [0, 90, 180, 270],
    "nadir_flight": true,  // 是否包含正射航线
    "flight_speed": 6.0
  },
  "aircraft_type": "M300_RTK"
}

4. 航带飞行任务规划 (plan_strip_mission)

规划线性航带飞行任务，适用于走廊巡检、管道巡查、道路巡查等线性目标的检查场景。

航带类型

管道巡查 (pipeline)

专门用于石油、天然气管道的巡检，支持管道走向跟踪

电力线巡检 (powerline)

电力线路巡检，支持杆塔定位和线路跟踪

道路巡查 (road)

道路状况检查，支持道路中心线跟踪

河流巡查 (river)

河流水质监测和河道变化检查

飞行模式

模式 描述 适用场景
single_line 单线飞行 窄带目标，如管道
parallel_lines 平行线飞行 宽带目标，如高速公路
zigzag 之字形飞行 需要详细检查的区域
back_and_forth 往返飞行 需要多角度观察的目标

5. 航线路径优化 (optimize_route)

优化航点顺序以最小化飞行距离、时间或能耗。支持多种优化算法和目标函数。

优化算法对比

算法 时间复杂度 优化质量 适用场景
nearest_neighbor O(n²) 中等 快速优化，航点数量较少
two_opt O(n²) 良好 平衡速度和质量
genetic_algorithm O(g×p×n) 优秀 复杂约束，高质量要求
simulated_annealing O(i×n) 优秀 大规模问题
christofides O(n³) 最优 小规模高精度要求

优化目标

minimize_distance

最小化总飞行距离，适用于节省时间的场景

minimize_time

最小化总飞行时间，考虑加速减速过程

minimize_energy

最小化能耗，延长飞行时间

balanced

平衡优化，综合考虑距离、时间和能耗

9. 辅助功能工具 (utility_functions)

提供坐标转换、距离计算、航线分析等辅助功能，支持多种坐标系统和计算方法。

坐标系统支持

坐标系统 代码 适用地区 说明
WGS84 WGS84 全球 GPS原始坐标系
GCJ02 GCJ02 中国大陆 国测局坐标系
BD09 BD09 中国大陆 百度坐标系
UTM UTM 全球分带 通用横轴墨卡托投影

功能列表

坐标转换

{
  "function_type": "convert_coordinates",
  "coordinates": [
    {"latitude": 39.9042, "longitude": 116.4074}
  ],
  "source_system": "WGS84",
  "target_system": "GCJ02"
}

距离计算

{
  "function_type": "calculate_distance",
  "points": [
    {"latitude": 39.9042, "longitude": 116.4074},
    {"latitude": 39.9142, "longitude": 116.4174}
  ],
  "unit": "meters",
  "calculation_method": "haversine"
}

航线分析

{
  "function_type": "analyze_flight_plan",
  "waypoints": [...],
  "flight_speed": 5.0,
  "include_detailed_stats": true,
  "analyze_efficiency": true
}

10. KMZ文件生成 (generate_kmz)

生成符合WPML标准的KMZ航线文件，可直接导入DJI Pilot 2使用。支持完整的航线信息和元数据。

WPML标准支持

  • ✅ 航点信息 (坐标、高度、动作)
  • ✅ 飞行参数 (速度、转弯模式、航向模式)
  • ✅ 安全设置 (失控动作、低电量动作)
  • ✅ 负载配置 (云台设置、相机参数)
  • ✅ 元数据 (作者、创建时间、版本)
  • ✅ 模板文件 (template.kml)

文件结构

mission.kmz
├── waylines.wpml          # 主航线文件
├── template.kml           # 模板文件
└── res/                   # 资源文件夹
    ├── icon.png          # 图标文件
    └── ...               # 其他资源

生成配置

{
  "flight_plan": {
    "waypoints": [...],
    "flight_speed": 5.0,
    "aircraft_type": "M30",
    "flight_params": {
      "heading_mode": "follow_wayline",
      "turn_mode": "to_point_stop",
      "gimbal_pitch": -90
    }
  },
  "output_filename": "mission.kmz",
  "include_template": true,
  "include_resources": false,
  "author": "DJI Waypoint Planner",
  "metadata": {
    "mission_name": "测绘任务_001",
    "description": "建图航拍任务",
    "created_date": "2024-07-20"
  }
}
